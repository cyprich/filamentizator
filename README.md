# Filamentizator

System to manage filaments for 3D printing

Kind of combination of my previous projects:

- [filamenty](https://github.com/cyprich/filamenty)
- [filamenty-rs](https://github.com/cyprich/filamenty-rs)

This project also includes `device`  
This is an IoT device, which connects to the API and displays filaments and stuff  
See more info in it's [README](./device/README.md) file
